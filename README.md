# react-quizz
Contexte : Cours sur les fondamentaux de React en H3, intervenant Arnoult

## Ce projet c'est quoi ?

Ceci est un projet de cours fait par Mathys GIRARDOT et Abiola Barr AKOBI, ayant pour objectif de maîtriser les bases de React.  
Celui-ci est donc un jeu, plus précisément un 2048, avec quelques petites fonctionnalités bonus par rapport au jeu original.

## Le jeu

### Comment lancer le jeu ?

### Il faut faire quoi pour gagner ?

* L'objectif est d'atteindre la somme de 2048 sur l'une des cases  
* Cependant, il faut faire attention car à chaque mouvement une nouvelle pièce apparaît sur le tableau  
* Si les pièces recouvrent la totalité du tableau et que vous ne pouvez plus rien déplacer, vous avez donc perdu

**Bonne chance !**

### Comment faire pour jouer

* Il suffit tout simplement d'utiliser les flèches en bas à droite de votre clavier pour déplacer les pièces dans le tableau devant vous

**À vous de jouer !**
